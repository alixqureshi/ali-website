Download the skill

This is a Claude skill. Download it, then install one of two ways below.

↓ Download SKILL.md

Option 1 — use it in regular chat (easiest)

Claude Desktop

Settings → Capabilities → Skills → + Add skill → drop in the SKILL.md → toggle it on. New chat: "build me an advertorial for [your URL]".

Claude.ai (no install)

Projects → Create project → upload SKILL.md as project knowledge (or paste its contents into the project instructions). Start a chat inside that project: "build me an advertorial for [your URL]".

ChatGPT, Cursor, or anything else

Open the downloaded SKILL.md, copy everything, paste it into Custom Instructions / System Prompt / Custom GPT instructions. Then: "build me an advertorial for [your URL]".

Option 2 — install in Claude Code (more flexibility)

Gives the skill full file-system access and multi-step tool use — it can save the HTML to your machine and iterate without copy-pasting.

  • Mac: Finder → Cmd+Shift+G → ~/.claude/skills/ → create a folder advertorial-builder-free → move SKILL.md into it → restart Claude Code.
  • Windows: same, but the folder is %USERPROFILE%\.claude\skills\.

Hosting the page

Once your AI hands you index.html:

  • Free in 30 seconds: drag the file onto app.netlify.com/drop — live on a .netlify.app subdomain instantly.
  • Custom domain: Cloudflare Pages or GitHub Pages, one-time DNS setup.

Two things to check before you run ads

  • Replace the Pixel placeholder. The HTML contains PIXEL_ID_HERE — search and replace with your real Meta Pixel ID, or no PageView / InitiateCheckout events fire.
  • Click every CTA. Open the file and click every button — they should all link to your real checkout / signup URL. This is how most beginners quietly ship a broken page.

Five-minute check. Saves a wasted ad-spend cycle.
